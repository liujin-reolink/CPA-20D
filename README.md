# CPA-20D

> 中国注册会计师（CPA）20 天冲刺复习项目。
>
> 目标不是替代教材，而是把六门专业阶段科目压缩成一套适合考前快速复习的 Markdown 作战手册。

## 项目目标

本项目计划围绕 CPA 专业阶段 6 门科目，整理一套结构统一、便于维护、适合打印和在线阅读的速成笔记。

六门科目包括：

- 会计
- 审计
- 财务成本管理
- 税法
- 经济法
- 公司战略与风险管理

每门科目计划整理为 20 个 Markdown 页面，总计约 120 个页面。

```text
6 门科目 × 20 页 = 120 页 A4 冲刺手册
```

## 设计原则

本项目不是传统教材摘抄，而是面向考试得分的冲刺资料。

每个页面尽量遵循以下原则：

1. 只保留高频考点和关键理解路径。
2. 优先覆盖真题中反复出现的知识点。
3. 使用表格、流程图、清单和口诀降低记忆成本。
4. 每页控制为一个独立主题，方便打印、复习和后续更新。
5. 内容以 Markdown 编写，适合 GitHub、VS Code、Obsidian 等工具阅读。

## 科目规划

| 目录 | 科目 | 重点方向 |
|---|---|---|
| `accounting/` | 会计 | 长投、合并、收入、金融工具、所得税 |
| `auditing/` | 审计 | 审计流程、风险评估、认定、审计证据、审计报告 |
| `financial-management/` | 财务成本管理 | 时间价值、资本成本、投资决策、本量利、成本管理 |
| `tax-law/` | 税法 | 增值税、企业所得税、个人所得税、消费税 |
| `economic-law/` | 经济法 | 公司法、证券法、合同编、破产法、票据法 |
| `strategy/` | 公司战略与风险管理 | SWOT、PEST、五力模型、价值链、风险管理 |

## 页面模板

每个知识点页面建议采用统一结构：

```markdown
# 页面标题

## 一句话理解

## 高频考点

## 必背规则

## 易错陷阱

## 真题套路

## 速记口诀

## 最后一分钟
```

## 适用方式

建议使用方式：

1. 先阅读 `ROADMAP.md`，确定 20 天学习节奏。
2. 每天按科目阅读对应 Markdown 页面。
3. 每学完一个主题，配套做近年真题或章节题。
4. 考前只看每页的“最后一分钟”和“易错陷阱”。

## 注意事项

- CPA 考试内容会随年度教材、准则和法规变化而变化。
- 本项目适合作为冲刺复习资料，不建议作为唯一学习来源。
- 税法、经济法等科目尤其需要结合当年最新教材和法规更新。
