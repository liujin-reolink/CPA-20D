# 审计

> 审计的关键不是背概念，而是理解注册会计师从接受业务到出具报告的完整流程。

## 冲刺重点

| 优先级 | 内容 |
|---|---|
| S | 审计流程、认定、重大错报风险、审计意见 |
| A | 风险评估、内部控制、控制测试、实质性程序 |
| B | 审计证据、审计抽样、函证、存货监盘 |
| C | 职业道德、质量管理、特殊项目审计 |
