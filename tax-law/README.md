# 税法

> 税法冲刺的核心是掌握主要税种的计算规则，尤其是增值税、企业所得税和个人所得税。

## 冲刺重点

| 优先级 | 内容 |
|---|---|
| S | 增值税、企业所得税、个人所得税 |
| A | 消费税、土地增值税、税收征管 |
| B | 资源税、房产税、城镇土地使用税、契税 |
| C | 印花税、车船税、环境保护税等小税种 |
