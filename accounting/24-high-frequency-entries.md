# 第二十四章 高频分录汇总

> 重要程度：⭐⭐⭐⭐⭐  
> 预计分值：不单独计分，但直接影响做题速度  
> 建议学习时间：1~2 小时  

---

## 为什么会考这一章？

会计考试最后一定会落到分录。

但是冲刺阶段不建议死背所有分录。

更有效的方法是：先理解业务，再记住高频分录模板。

本章不是完整分录大全，而是把最常见、最容易在考试中出现的分录集中放在一起，方便考前快速回看。

---

## 一句话理解

> 分录不是目的，分录只是把业务翻译成会计语言。

---

## 核心知识

### 1. 收入相关分录

确认收入：

```text
借：银行存款 / 应收账款 / 合同资产
贷：主营业务收入
    应交税费——应交增值税（销项税额）
```

结转成本：

```text
借：主营业务成本
贷：库存商品
```

预收款但尚未履约：

```text
借：银行存款
贷：合同负债
```

履约后确认收入：

```text
借：合同负债
贷：主营业务收入
```

### 2. 存货跌价

计提存货跌价准备：

```text
借：资产减值损失
贷：存货跌价准备
```

转回跌价准备：

```text
借：存货跌价准备
贷：资产减值损失
```

### 3. 固定资产

购入固定资产：

```text
借：固定资产
贷：银行存款 / 应付账款
```

计提折旧：

```text
借：制造费用 / 管理费用 / 销售费用
贷：累计折旧
```

处置固定资产转入清理：

```text
借：固定资产清理
    累计折旧
    固定资产减值准备
贷：固定资产
```

### 4. 无形资产

购入无形资产：

```text
借：无形资产
贷：银行存款
```

摊销无形资产：

```text
借：管理费用 / 制造费用等
贷：累计摊销
```

研究阶段支出：

```text
借：研发支出——费用化支出
贷：银行存款 / 应付职工薪酬等
```

期末转入损益：

```text
借：管理费用
贷：研发支出——费用化支出
```

开发阶段符合资本化条件：

```text
借：研发支出——资本化支出
贷：银行存款 / 应付职工薪酬等
```

达到预定用途：

```text
借：无形资产
贷：研发支出——资本化支出
```

### 5. 长期股权投资

取得长期股权投资：

```text
借：长期股权投资
贷：银行存款等
```

权益法下确认投资收益：

```text
借：长期股权投资——损益调整
贷：投资收益
```

权益法下被投资单位亏损：

```text
借：投资收益
贷：长期股权投资——损益调整
```

权益法下宣告现金股利：

```text
借：应收股利
贷：长期股权投资——损益调整
```

成本法下宣告现金股利：

```text
借：应收股利
贷：投资收益
```

### 6. 金融工具

交易性金融资产取得：

```text
借：交易性金融资产——成本
    投资收益（交易费用）
贷：银行存款
```

公允价值上升：

```text
借：交易性金融资产——公允价值变动
贷：公允价值变动损益
```

公允价值下降：

```text
借：公允价值变动损益
贷：交易性金融资产——公允价值变动
```

债权投资确认利息收益：

```text
借：应收利息
    债权投资——利息调整（或贷方）
贷：投资收益
```

### 7. 借款费用

一般费用化利息：

```text
借：财务费用
贷：应付利息 / 银行存款
```

符合资本化条件的利息：

```text
借：在建工程 / 存货等
贷：应付利息 / 银行存款
```

### 8. 所得税

确认当期所得税：

```text
借：所得税费用
贷：应交税费——应交所得税
```

确认递延所得税负债：

```text
借：所得税费用
贷：递延所得税负债
```

确认递延所得税资产：

```text
借：递延所得税资产
贷：所得税费用
```

### 9. 或有事项

确认预计负债：

```text
借：销售费用 / 管理费用 / 营业外支出等
贷：预计负债
```

实际支付时：

```text
借：预计负债
贷：银行存款
```

### 10. 政府补助

收到与收益相关补助，补偿已发生费用：

```text
借：银行存款
贷：其他收益 / 营业外收入
```

收到与资产相关补助，总额法：

```text
借：银行存款
贷：递延收益
```

分期计入损益：

```text
借：递延收益
贷：其他收益 / 营业外收入
```

## 考试重点

| 重点 | 怎么考 | 复习建议 |
|---|---|---|
| 收入确认和成本结转 | 综合题基础 | 必须熟练 |
| 权益法长投 | 投资收益、股利、其他综合收益 | 必须熟练 |
| 金融资产公允价值变动 | 进损益还是其他综合收益 | 高频 |
| 递延所得税 | DTA、DTL确认方向 | 高频 |
| 固定资产处置 | 账面价值和清理 | 基础 |

---

## 容易混淆

| 易错点 | 正确理解 |
|---|---|
| 权益法收到股利确认投资收益 | 通常冲减长期股权投资 |
| 成本法下子公司盈利确认投资收益 | 个别报表成本法下通常不确认 |
| 交易性金融资产交易费用进成本 | 通常计入投资收益 |
| 折旧费用一定进管理费用 | 要看资产用途 |
| 递延所得税资产和负债方向死背 | 先判断暂时性差异性质 |

---

## 真题套路

真题通常不会单独问“写分录”。

它会给业务背景，让你计算金额、判断报表影响，分录只是背后的逻辑。

复习时不要孤立背分录，而要配合章节理解：

- 收入分录配合收入确认。
- 长投分录配合成本法、权益法。
- 所得税分录配合暂时性差异。
- 固定资产分录配合折旧、减值、处置。

---

## 一句话总结

> 高频分录背的是业务逻辑，不是借贷方向本身。

---

## 考前一分钟

- 收入确认后通常还要结转成本。
- 权益法下，被投资单位赚亏影响长投账面价值。
- 权益法下现金股利通常冲减长投。
- 交易性金融资产交易费用通常进投资收益。
- 固定资产折旧进哪里，看资产用途。
- 递延所得税先判断暂时性差异，再写分录。
