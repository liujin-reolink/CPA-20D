# 会计

> CPA 六科中的核心科目，也是审计、税法和财管部分理解的基础。

## 冲刺重点

| 优先级 | 内容 |
|---|---|
| S | 长期股权投资、企业合并、合并财务报表 |
| A | 收入、金融工具、所得税、租赁 |
| B | 资产减值、存货、固定资产、无形资产 |
| C | 政府补助、外币折算、债务重组、非货币性资产交换 |
