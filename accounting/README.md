# 会计

> CPA 专业阶段的核心科目，也是审计、税法和财管部分理解的基础。

## 使用说明

本目录是《会计》冲刺手册 V0.1。

目标不是替代教材，而是在考前有限时间内帮助你快速建立主线、抓住高频章节、避开常见坑。

建议先看核心主线章节，再看基础章节，最后看高频分录和考前速记。

## 冲刺重点

| 优先级 | 内容 | 学习建议 |
|---|---|---|
| S | 长期股权投资、企业合并、合并财务报表 | 必须优先学 |
| A | 收入、金融工具、所得税、租赁 | 高频，建议熟练 |
| B | 资产减值、存货、固定资产、无形资产、投资性房地产 | 掌握基础和易错点 |
| C | 政府补助、外币折算、债务重组、非货币性资产交换 | 时间紧时看考前一分钟 |
