# 财务成本管理

> 财管是公式和计算驱动科目，冲刺阶段要优先掌握公式适用场景，而不是推导过程。

## 冲刺重点

| 优先级 | 内容 |
|---|---|
| S | 时间价值、资本成本、投资项目评价、本量利分析 |
| A | 杠杆、营运资本、标准成本、预算 |
| B | 企业价值评估、期权、业绩评价 |
| C | 长期筹资、股利分配、作业成本 |
